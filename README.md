# React Firebase Portfolio Project

This project is a portfolio website built with React and TypeScript, integrated with Firebase for backend services and Cloudinary for media management.

## Features
- Multiple pages including Home, Projects, Contact, Admin Dashboard, and AI Fit Analysis
- Protected routes for admin functionalities
- Cloudinary integration for image and media resources
- Firebase authentication and data management

## Prerequisites
- Node.js installed on your machine
- Firebase project setup with appropriate credentials
- Cloudinary account for media management

## Setup and Run Locally

1. Install dependencies:
   ```
   npm install
   ```

2. Set up environment variables:
   - Configure Firebase and Cloudinary credentials in the respective config files or environment variables.

3. Run the app:
   ```
   npm run dev
   ```
